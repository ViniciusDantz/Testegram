A análise valor limite é uma técnica de teste de caixa preta, isso basicamente significa que o testador não terá como base de teste o código do software e sim a interface em execução do software. Esta técnica tem por objetivo a identificação de defeitos nos valores limites de dados numéricos/sequenciais de entrada/saída do software, mostrando defeitos que podem por vezes passar despercebidos por desenvolvedores. A técnica pode ter variações, duas dessas variações são:

  • Dois pontos, que se refere a escolher dois valores para cada valor limite;

  • Três pontos, que se refere a escolher três valores para cada valor limite.
